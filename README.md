## ⚙️ Tech Stack
- Frontend Framework: React, Next.js <br>
- Styling: Tailwind CSS <br>
- State Management: React Query, WebSocket, Context API <br>
- Data Fetching: Binance API (REST & WebSocket) <br>
- Charting Library: ApexCharts <br>
- Others: ESLint <br>
